# components

Reusable UI and page sections.

## Folders

- **layout** — Header, Footer, shell components.
- **sections** — Page-specific sections (Hero, ServicesOverview, WhyChooseUs, etc.).
- **ui** — Buttons, inputs, cards, icons. Small, reusable, Tailwind-only.

## Conventions

- No inline styles; Tailwind only.
- No magic numbers; use theme or shared constants.
- Icons: custom SVG components in `ui/icons` or assets in `public/svg`.
