# ui

Reusable UI primitives: buttons, inputs, cards, icons.

## Structure

- **icons/** — Custom SVG React components (SteelBeamIcon, ColumnIcon, etc.). Minimal, geometric.
- Buttons, inputs, cards at this level.

## Conventions

- No inline styles; Tailwind only.
- Export from index or direct import by name.
- Icons: no emoji; SVG only.
