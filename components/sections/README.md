# sections

Page-specific section components composed into app routes.

## Examples

- Hero, ServicesOverview, ServiceArea, WhyChooseUs, ProjectGalleryPreview, CtaSection (Home).
- Service section blocks for Services page.
- Project card grid for Projects page.

## Conventions

- One section per file when practical.
- Accept optional className; use `cn()` for merging.
- Prefer Server Components; use client only when section has interactivity.
