# supabase

Supabase client scaffold for mwfab.co.

## Files

- **client.ts** — Browser client; use in Client Components.
- **server.ts** — Server client with cookie handling; use in Server Components and API routes.

## Env

- `NEXT_PUBLIC_SUPABASE_URL`
- `NEXT_PUBLIC_SUPABASE_ANON_KEY`

## Future

Schema and migrations will live in `/supabase/schema/`, versioned and commented. See `/docs/future-schema.md`.
