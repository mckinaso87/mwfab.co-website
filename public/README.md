# public

Static assets.

## Folders

- **images/** — General images.
- **images/projects/** — Project gallery placeholders.
- **images/logo/** — Logo (placeholder for metallic logo on dark backgrounds).
- **svg/** — Custom SVG assets (steel beam, column, welding spark, structural frame).

## Conventions

- Use `next/image` in app; set width/height or sizes.
- Placeholder images here are temporary; document in README. Replace with real assets before production.
